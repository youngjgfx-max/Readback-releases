# Readback — Releases

Este repositorio solo contiene el feed de actualizaciones (`appcast.xml`) y los
instaladores (`.dmg`) de cada versión de [Readback](https://github.com/youngjgfx-max/Readback),
publicados como GitHub Releases.

El código fuente de la app vive en un repositorio privado aparte. Este repo es
público únicamente para que el auto-actualizador (Sparkle) pueda leer
`appcast.xml` sin autenticación.
